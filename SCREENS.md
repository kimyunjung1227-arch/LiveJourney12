## 1. 홈

**진입**: 앱 첫 진입 시 기본 화면. 하단 탭의 첫 번째.

**구조**:
- 상단 헤더 (높이 56px)
  - 좌측: "Live Journey" 로고 (font-size: 17px, color: #4DB8E8, weight: 600)
  - 우측: 검색 아이콘 + 알림 아이콘 (gap: 14px, size: 21px)
- 라이브 인디케이터 박스 (배경 #E8F4FB, padding: 14px 18px)
  - 키컬러 점 + 글로우 + "지금 47장 라이브" (color: #1A6EA8)
- 카테고리 필터 (가로 스크롤, 칩 형태)
  - "전체" + 카테고리 3개 + "+3" 더보기 칩
- 게시물 피드 (시간순)
  - 카드 사이 구분선 또는 배경 회색 영역

**일반 게시물 카드 구조**:
1. 사진 (높이 280px, border-radius: 12px)
   - 좌상단 EXIF 인증 뱃지
   - 우상단 카테고리 뱃지 (흰 배경 + 아이콘 + 라벨)
2. 작성자 영역 (margin-bottom: 12px)
   - 아바타 (30px 솔리드 키컬러)
   - 이름 (font-size: 14px, weight: 600)
   - 지금 현장 뱃지 (해당 시)
3. 위치 + 남은 시간 (font-size: 12px, color: #6B6B6B)
   - 핀 아이콘 + 장소명 + "·" + "47시간 남음"
4. 본문 (font-size: 14px, line-height: 1.6)
5. 반응 지표 (구분선 위, padding-top: 14px)
   - 좋아요 / 저장 / 갈게요 (갈게요는 키컬러)

**베스트 컷 카드 구조** (일반 카드와 시각 차별):
- 카드 전체에 키컬러 테두리 1.5px
- 상단에 그라데이션 헤더 뱃지 "여의도 한강공원 베스트 컷"
- 작성자 이름 옆 "베스트 컷 작가" 그라데이션 라벨
- 반응 지표가 키컬러로 채워짐 (좋아요 187 키컬러, 갈게요 42 키컬러)

**묶음 카드 구조** (같은 작성자가 1시간 이내 같은 장소에 여러 장):
- 사진 위 중앙에 흰 막대 인디케이터 (3개 막대, 첫 번째 활성)
- 사진 위 우하단에 "1/3" 표시
- 좌우 스와이프로 사진 전환

**인터랙션**:
- 사진 탭 → 풀스크린
- 작성자 아바타/이름 탭 → 다른 사용자 프로필
- 위치명 탭 → 장소 페이지
- 카테고리 뱃지 탭 → 카테고리 화면

## 2. 핫플

**진입**: 하단 탭 두 번째.

**구조**:
- 헤더 (높이 56px)
  - 좌측: "실시간 핫플" (font-size: 17px, weight: 600)
  - 우측: 검색 아이콘
- 라이브 인디케이터 박스 (#E8F4FB)
  - "최근 6시간 활동 기준"
- **1~3위 강조 카드 영역**
- **4~20위 사진 3장 리스트 영역**
- 하단 "전체 보기" 버튼 (서브틀 버튼)

**1~3위 강조 카드 구조**:
1위 카드 (가장 큼):
- 사진 240px (border-radius: 12px)
  - 우상단 검정 박스에 "🔥 HOT 1위"
- 장소명 + 도시 (font-size: 16px, weight: 700)
- "32장 · ↑ 187%" (font-size: 12px, color: #6B6B6B)
- 통계 박스 (회색 배경)
  - "지금 12명 보는 중" / "최근 6시간 32장"

2위 카드 (사진 200px), 3위 카드 (사진 180px) 동일 구조 점진적 축소.

**4~20위 리스트 구조**:
각 항목:
- 좌측 순위 번호 (font-size: 14px, weight: 700, width: 18px)
- 중앙 정보 (flex: 1)
  - 장소명 (font-size: 13px, weight: 600)
  - 도시 · N장 · ↑N% (font-size: 10px, color: #6B6B6B)
- 우측 사진 3장 가로 (각 사진 56px, EXIF 시간 좌상단)

**인터랙션**:
- 1~3위 카드 또는 4~20위 행 탭 → **장소 페이지로 진입** (베스트 컷 히어로 영역 노출)

## 3. 카메라

**진입**: 하단 탭 가운데 (가장 큰 진입). 또는 어떤 화면에서든 "사진 올리기" 버튼.

**구조**:
- 다크 UI (배경 #0A0A0A)
- 헤더 (padding: 6px 8px)
  - 좌측: X 닫기 아이콘 (흰색)
  - 우측: 플래시 아이콘
- EXIF 안내 박스 (반투명 키컬러 배경)
  - 방패 아이콘 + "EXIF 자동 인증" (color: #4DB8E8)
- 카메라 뷰파인더 (큰 영역, aspect-ratio: 3/4)
- 사진/영상 토글
- 셔터 버튼 (중앙, 흰색 원 + 키컬러 테두리)
- GPS 위치 박스 (하단)
  - 핀 아이콘 + "여의도 한강공원" (자동 인식)

**인터랙션**:
- 셔터 탭 → 정보 입력 화면으로 이동
- 영상 모드 토글 시 셔터 길게 눌러 녹화

**중요**: 사진 갤러리에서 가져오기 옵션 없음. **카메라 앱에서 직접 촬영만** EXIF 인증 유지.

## 5. 프로필 (내 프로필)

**진입**: 하단 탭 마지막.

**구조**:
- 헤더
  - 좌측: "프로필" (font-size: 17px, weight: 600)
  - 우측: 설정 아이콘
- 프로필 영역
  - 솔리드 키컬러 아바타 (64px)
  - **베스트 컷 작가면 우하단에 작은 왕관 인디케이터** (22px, 그라데이션, 흰 테두리)
  - 이름 (font-size: 16px, weight: 600)
  - **베스트 컷 작가면 이름 옆에 그라데이션 라벨**
  - "라이브저니 N번째 여행자 · 가입 N개월"
  - EXIF 100% / 오늘 활동 뱃지 (키컬러 옅은 배경)
- **통계 4개 그리드** (베스트 컷 있을 때):
  - 7일 활동 (회색 배경)
  - 도움 받은 사람 (회색 배경)
  - 갈게요 받음 (회색 배경, 키컬러 강조)
  - **베스트 컷 (그라데이션 배경, 키컬러 강조)**
- **대표 베스트 컷 캐러셀** (베스트 컷 있을 때):
  - 헤더: 왕관 + "대표 베스트 컷 N" + 전체 보기 링크
  - 가로 스크롤, 각 사진 120px
  - 각 사진: 키컬러 테두리 + 우상단 왕관 뱃지 + 하단 장소명 + "도움 N명 받음"
- 지금 라이브 영역
  - 라이브 인디케이터 + "지금 라이브 N"
  - 키컬러 테두리 사진 2~3장
- 기록 아카이브
  - 시즌 필터 칩 (2026 봄 / 2025 가을 / 도시별)
  - 사진 그리드 3열
- 시즌 데이터 카드 (#E8F4FB 배경)
  - "작년 봄 N님은" + 작년 같은 시기 활동 요약

**인터랙션**:
- 베스트 컷 사진 탭 → 풀스크린
- 베스트 컷 작가 라벨 탭 → 자기 프로필 (자기 영예 페이지)

## 6. 검색

**진입**: 홈 우상단 검색 아이콘, 핫플 우상단 검색 아이콘.

**구조**:
- 헤더 (높이 56px)
  - 좌측: 뒤로 가기
  - 검색바 (#F5F7FA 배경, border-radius: 20px)
- 진입 카드 2개 (그리드 2x1, 키컬러 옅은 배경)
  - 실시간 질문 카드 (질문 아이콘 + 3 대기 뱃지)
  - 시즌 캘린더 카드 (캘린더 아이콘 + 2주 후 뱃지)
- 자주 보는 도시 (북마크한 도시)
  - 칩 형태 (키컬러 옅은 배경 + 키컬러 테두리)
- 실시간 핫플 미리보기
  - 헤더 + "전체 보기 →" 링크
  - 1~3위 행 (회색 배경)
- 카테고리 그리드 (6개, 2x3)
  - 각 카테고리: 회색 배경 + 아이콘 + 라벨 + "지금 N곳" (키컬러)

**검색어 입력 시**:
- 검색 결과 화면 전환
- 도시 결과 (키컬러 카드, "라이브 N장 보기" 버튼)

## 8. 알림

**진입**: 홈/핫플 우상단 알림 아이콘.

**구조**:
- 헤더
  - 좌측: 뒤로 + "알림"
  - 우측: 모두 읽음 체크 아이콘
- 필터 칩 (가로 스크롤)
  - 전체 (활성) / 영향력 / 관심 도시 / 베스트 컷
- 알림 리스트

### 일반 알림 카드 구조

좌측 원형 아이콘 (40px, 회색 또는 키컬러 옅은 배경)
- 영향력 알림: 핀/하트 아이콘, 키컬러 옅은 배경
- 일반 알림: 회색 배경
중앙 정보:
- 내용 (font-size: 13px, line-height: 1.5)
- 시간 (font-size: 10px, color: #6B6B6B)
우측 미리보기 썸네일 (44px) 또는 라이브 점

### 베스트 컷 영예 알림 (특별 카드)

- 키컬러 그라데이션 부드러운 배경 (`linear-gradient(135deg, rgba(77,184,232,0.08), rgba(77,184,232,0.18))`)
- 키컬러 테두리 1.5px
- 우상단에 큰 왕관 장식 (font-size: 90px, opacity: 0.15)
- 그라데이션 작은 뱃지 "새 베스트 컷"
- 큰 헤더 카피 (font-size: 17px, weight: 700, line-height: 1.4)
  - "축하해요! 여의도 한강공원 베스트 컷이 됐어요"
- 통계 설명 (font-size: 12px, color: #1A6EA8)
  - "187명이 좋아요, 73명이 저장, 42명이 갈게요 표시한 사진"
- 미니 사진 카드 (흰 배경, padding: 12px)
  - 좌측 사진 56px + 왕관 뱃지
  - 본문 미리보기 + "N시간 전 올린 사진"
  - 우측 화살표

**인터랙션**:
- 일반 알림 탭 → 해당 사진 풀스크린 또는 장소 페이지
- 베스트 컷 영예 알림 탭 → 그 장소 페이지

## 9. 정보 입력

**구조**:
- 헤더
  - 좌측: 뒤로 + "정보 입력"
  - 우측: "업로드" 버튼 (키컬러)
- 사진 미리보기 (280px)
  - 좌상단 EXIF 자동 인증 박스 (반투명 검정)
    - 방패 아이콘 + "EXIF 자동 인증" + "방금 · 오후 4:23"
  - 좌하단 GPS 위치 박스 (반투명 검정)
    - 핀 아이콘 + "여의도 한강공원"
- 카테고리 선택 (6개 그리드, 필수)
  - 선택: 키컬러 옅은 배경 + 키컬러 테두리
  - 비선택: 흰 배경 + 회색 테두리
- 한 줄 설명 (선택)
  - 회색 배경 박스 (min-height: 60px)

## 베스트 컷 시스템 화면 흐름 요약 ⭐

베스트 컷 시스템은 라이브저니 핵심 차별점. 4개 화면이 연결되어 작동.

```
[사용자가 사진 업로드]
        ↓
[홈 피드에 일반 카드로 노출]
        ↓
[좋아요 + 갈게요 + 저장 누적]
        ↓
[베스트 컷 알고리즘이 선정]
(좋아요 + 갈게요 × 2 + 저장 × 1.5)
        ↓
[베스트 컷 영예 알림 푸시] ──→ [알림 화면의 그라데이션 영예 카드]
                                    │
                                    └─→ [장소 페이지의 베스트 컷 히어로 영역]
        ↓
[프로필에 베스트 컷 N+1로 누적]
[베스트 컷 작가 라벨 (그라데이션) 자동 표시]
[아바타 우하단 왕관 인디케이터]
        ↓
[홈 피드에서 그 사용자의 베스트 컷 카드 등장]
(키컬러 테두리 + 그라데이션 헤더)
        ↓
[다른 사용자가 그 카드 보고 작성자 프로필 진입]
        ↓
[대표 베스트 컷 캐러셀에서 다른 베스트 컷 확인]
        ↓
[팔로우 가능성 증가]
```

## 베스트 컷 관련 4개 화면 한눈에 정리

| 화면 | 베스트 컷 표현 방식 |
|---|---|
| **홈 피드** | 키컬러 테두리 카드 + 그라데이션 헤더 뱃지 + 작성자 옆 라벨 + 반응 지표 키컬러 |
| **장소 페이지** | 히어로 영역 320px 사진 + 그라데이션 작성자 정보 + 영예 트러스트 카드 |
| **알림 (영예)** | 그라데이션 큰 카드 + 왕관 장식 + 축하 헤더 + 미니 사진 |
| **프로필** | 아바타 우하단 왕관 + 이름 옆 라벨 + 통계 그라데이션 + 대표 캐러셀 |

## 참고 사항

### 디자인 시스템과의 관계
이 문서는 화면 구조 명세서. **컬러, 폰트, 간격, 컴포넌트 세부**는 CLAUDE.md 참조.

### 베스트 컷 알고리즘 구현
서버 사이드 로직. 각 사진에 대해:
```
score = likes + (going_count × 2) + (save_count × 1.5)
```
가장 높은 점수의 사진이 그 장소의 베스트 컷. 점수가 동일하면 EXIF 시간 최신 우선.

베스트 컷 변경 시 이전 베스트 컷 작가에게도 알림 ("회수" 알림은 보내지 않음. 새로 선정된 작가에게만 영예 알림).

### 베스트 컷 변경 주기
실시간 (매 반응마다 즉시 재계산). 단, 알림은 처음 베스트 컷이 됐을 때 1회만.

48시간 후 라이브에서 내려가면 그 시점의 베스트 컷이 그 장소의 그 기간 대표로 영구 기록.

### 자영업자 매장 베스트 컷
일반 장소와 별도 산정. 매장 페이지 내에서만 영구 보관. 시즌별로 자동 누적.
